jQuery Simple Carousel Slider

Simple jQuery Carousel Slider Plugin

Installation

bower install jquery.simple-carousel-slider

Usage

  1. Include jQuery and jquery.finger.

    <script src="../lib/jquery/jquery.js"></script>
<script src="../lib/jquery.finger/jquery.finger.js"></script>

  2. Include plugin's code:

    <link rel="stylesheet" href="../src/css/jquery.simple-carousel-slider.css"> 
<script src="../src/js/jquery.simple-carousel-slider.js"></script>

  3. Call the plugin:

     <style>
   .sc-slider{
     margin: 0 auto;
     width:  500px;
     height: 333px;
   }
 </style>

 <div class="sc-slider">
   <ul class="sc-slides">
     <li class="sc-slide">
       <a href="#">
         <img src="images/img1.png" />
       </a>
     </li>
     <li class="sc-slide">
       <a href="#">
         <img src="images/img2.png" />
       </a>
     </li>
     <li class="sc-slide">
       <a href="#">
         <img src="images/img3.png" />
       </a>
     </li>
   </ul>
 </div>

 <script>
   $(function(){

     $('.sc-slider').scSlider({
       interval: 5000,
       prev: "＜",
       next: "＞"
     });

   });
 </script>
